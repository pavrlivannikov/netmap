NetMap v0.1 — Карта сети
=========================

Запуск:
  Просто откройте NetMap.exe

Требования:
  - Windows 10 или новее
  - WebView2 (есть в Windows 10+, если нет — установится автоматически)
  - Для ARP-сканирования: Npcap (https://npcap.com)

Что умеет:
  - Обнаружение устройств в локальной сети (ARP, ICMP, TCP)
  - Интерактивная карта сети с авто-раскладкой
  - Свои иконки для разных типов устройств
  - Определение производителя по MAC-адресу
  - Экспорт схемы в PNG и SVG

Горячие клавиши:
  Ctrl+S — начать сканирование
  Ctrl+L — сменить раскладку
  Ctrl+E — экспорт PNG
