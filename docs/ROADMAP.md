# NetMap — Roadmap

## v0.1 — Прототип (май 2026) ✅ DONE
- [x] Документация и спецификация
- [x] Структура проекта (Tauri + Svelte + Cytoscape.js)
- [x] SVG-иконки (8 типов устройств)
- [x] Базовый UI (Toolbar, Graph, DevicePanel, StatusBar)
