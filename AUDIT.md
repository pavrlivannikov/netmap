## 2026-05-25

---

## 1. ЦЕЛЬ ПРОЕКТА

NetMap — кросс-платформенный сетевой анализатор. Админ заходит в незнакомую сеть → 1 минута → видит:
- Какие устройства в сети (Discover)
- Как они связаны (Topology/LLDP)
- Где какой MAC воткнут (FDB)
- Что изменилось (Monitor/Diff)

**Стек:** Rust (бэкенд) + Svelte/Tauri (фронтенд) + SQLite (история)

## 3. ПРОБЛЕМЫ И БЛОКЕРЫ

### 🔴 Критические

1. **Сборка Rust на Windows невозможна**
   - MSVC: `link.exe` крашится (0xc0000005) — нужен repair VS 2022
   - GNU: нет `dlltool` — нужен полный MinGW (MSYS2), winget не работает
   - Кросс-компиляция: MinGW ломает UDP-сокеты → SNMP не работает

2. **PowerShell прототип нестабилен**
   - `$MyInvocation` = null при запуске через C# Process.Start → пути ломаются
   - `$env:TEMP` разный у C# и PowerShell
   - Полный скрипт виден в консоли (некрасиво)

### 🟡 Средние

3. **Нет нативной OUI-базы** — vendor определяется только на Linux (через nmap)
4. **FDB не работает на T2600G** — BRIDGE-MIB не поддерживается, нужен SSH-fallback
5. **LLDP-топология не протестирована** на живом оборудовании
